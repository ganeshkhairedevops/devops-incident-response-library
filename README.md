# devops-incident-response-library

This repository contains real-world DevOps troubleshooting scenarios and solutions encountered while working with infrastructure environments such as:

* Linux Servers
* Apache / Nginx Web Servers
* Kubernetes Clusters
* Jenkins CI/CD Pipelines
* Networking and Firewall Troubleshooting

The purpose of this repository is to maintain a documented knowledge base for quick incident resolution and DevOps learning.

## Categories

### Apache

Troubleshooting Apache service failures, port conflicts, and firewall issues.

### Nginx

SSL configuration, service failures, and web server deployment.

### Kubernetes

Deployment management, rolling updates, jobs, cronjobs, and replica sets.

### Jenkins

CI/CD pipeline configuration, plugin management, and node connectivity.

### Networking

Firewall rules, iptables configuration, and connectivity troubleshooting.

## Tools Used

* Linux (CentOS / RHEL)
* Nginx
* Apache
* Kubernetes
* Jenkins
* AWS
* iptables
